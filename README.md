# Stereo-poi

Web Audio APIを利用して、ステレオマイクのような挙動をブラウザ上で遊べるWebアプリケーションです。
マイク近くの発声を再現するため、左右の音量差を意図的に極端に設定しています。

> **開発の経緯**
> 元々はDiscordの通話で友人と立体音響で遊ぶ目的で開発しましたが、Discord側の音声処理によって当然モノラル化されてしまうため、当初の目的としては使えませんでした。

## 機能

- **マイク入力 / ファイル読み込み:** リアルタイムの音声入力、または既存のオーディオファイルを使用可能
- **一時停止 / 再生:** 読み込んだファイルの再生コントロール
- **録音機能:** 操作中の立体音響をそのままWAVファイルとして保存
- **音量調整 / ノイズ閾値調整:** スライダーでそれぞれ調整可能

## 使い方

1. 画面上のメニューから**マイク**または**ファイル**を選択します。
2. **録音ボタン（●）**を押して録音を開始します。
3. 画面下の**スピーカーアイコン**をマウスでドラッグして動かすことで、直感的にマイクに音を近づけたり遠ざけたりして遊べます。
4. 再度**録音ボタン**を押すと録音が完了し、`.wav` ファイルとしてPCにダウンロードされます。
